**AI Resume Critiquer**
AI Resume Critiquer is a Streamlit web application that allows users to upload their resumes and get AI-powered feedback tailored to their career goals and target job roles. The feedback focuses on content clarity, skills presentation, experience descriptions, and provides specific improvement suggestions.

**Features**
Upload resumes in PDF or TXT format.
Input your target job role for tailored feedback.
AI analysis using OpenAI's GPT-4o-mini model.
Structured, actionable recommendations to improve your resume.
NOTE: Provide your OpenAI API Key in env file before running the code

**Installation**
Clone the repository:
'''bash
git clone https://github.com/your-username/ai-resume-critiquer.git
cd ai-resume-critiquer
'''
Run the Code:
uv run streamlit run main.py
